Step 1: 选择合并方式

Join (Inner) 只保留满足匹配条件的行，并返回两表所有列
Left Join 保留左表的所有行 + 匹配上的右表的行（否则补 NULL），返回两表所有列
Right Join 保留右表的所有行+匹配上的左表的行(否则补NULL)，返回两表所有列
Union ALL 列数必须相同，不考虑列名是否相同，合并所有行，不去重

GENERATED SQL OUTPUT

query.sql 
SELECT
    t.transaction_id,
    t.customer_email,
    t.amount_usd,
    t.currency,
    b.billing_cycle,
    b.balance           AS current_balance
FROM
    v_user_billing_summarized_q4  t
INNER JOIN
    v_billing_q4                  b
    ON t.customer_email = b.customer_email
WHERE
    t.status = 'SUCCESS'
    AND b.balance > 0
ORDER BY
    t.amount_usd DESC;
